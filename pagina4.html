Servidor
Equipo o software que proporciona servicios a otros equipos o clientes en una red.
Red LAN
Red de área local que conecta dispositivos en un espacio geográfico limitado, como una oficina o aula.
IP (Protocolo de Internet)
Dirección única que identifica un dispositivo dentro de una red.
DNS (Sistema de Nombres de Dominio)
Servicio que traduce nombres de dominio legibles por humanos a direcciones IP.
Firewall
Dispositivo o software que controla y filtra el tráfico de red para proteger un sistema o red.
Virtualización
Tecnología que permite ejecutar varios sistemas operativos de forma simultánea en un mismo equipo físico.

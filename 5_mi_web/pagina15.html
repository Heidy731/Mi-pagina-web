1. Estructura básica de HTML

HTML es un lenguaje de marcado que permite estructurar contenido en la web. Toda página HTML tiene:

  • <!DOCTYPE html> - Define el tipo de documento.
  • <html> - Contenedor principal de la página.
  • <head> - Contiene metadatos, título
  • <body> - Contiene todo el contenido visible de la página.

2. Etiquetas comunes

Algunas etiquetas HTML comunes son:

  • <h1> a <h6> - Títulos y subtítulos.
  • <p> - Párrafos.
  • <a> - Enlaces.
  • <img> - Imágenes.
  • <ul> y <ol> - Listas.

3. Propiedades de texto

Para mejorar la presentación del texto, CSS ofrece:

  • font-weight: negrita o normal.
  • font-style: cursiva o normal.
  • text-transform: mayúsculas, minúsculas, capitalizar.
  • letter-spacing: espacio entre letras.
  • line-height: altura de línea.

4. Listas y enlaces

Las listas y enlaces son elementos importantes:

  • <ol> - Lista ordenada.
  • <ul> - Lista desordenada.
  • <a href="url"> - Enlaces a páginas o secciones internas.
Puedes enlazar a un tema específico usando id en los encabezados.
